# Getting Started

## Step 1: Instala los paquetes con NPM o Yarn

Ejecuta el siguiente comando

```sh
# Using npm
npm install

# OR using Yarn
yarn install
```

## Step 2: Levanta el app

Asegurate tener emulador android o iOS, o conectado via USB.

### Android

```sh
# Using npm
npm run android

# OR using Yarn
yarn android
```

### iOS

```sh
# Using npm
npm run ios

# OR using Yarn
yarn ios
```
